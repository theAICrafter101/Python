# Python
This repository is all about the python language from beginner to advance level.

#### Why to use Python?
Python is a:
- general purpose, easy to use, high-level, dynamically typed language.
- used in many fields, i.e., web development, data science, machine learning, automation, IoT, cybersecurity, scrippting, embedded systems and many more.

Following libraries/frameworks of Python are used in a specific domains:
- Pandas, Numpy → Data Analysis
- Scikit, Tensorflow → Machine Learning
- Django, Fast API, Flask → Web Development
- Selenium, BeautifulSoup → To interact with websites
